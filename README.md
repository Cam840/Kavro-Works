# Kavro Works 4-Page Website

This version keeps the site to exactly 4 HTML pages:

1. `index.html` — Home + About + quick service preview
2. `services.html` — Services, starting prices, and how it works
3. `work.html` — Gallery, reviews, and service area
4. `quote.html` — Quote form, availability calendar, and contact

Shared files:
- `styles.css` — all styling and mobile responsiveness
- `script.js` — hamburger menu, scroll reveal, carousel, and stats
- `images/LOGOKAVRO.png`
- `images/TOOLS_2D_NO_BG.png`

Upload all files and folders to the root of your GitHub Pages repository.

Important:
- Keep the `images` folder exactly named `images`.
- The quote form still uses your Formspree endpoint: `https://formspree.io/f/xykvyeka`.
- The calendar uses the Google Calendar embed URL already present in your latest file.
- File uploads are not included because Formspree requires an upgraded plan for attachments. The quote page tells customers to text photos instead.
- To add your own photo, upload it as `images/kamran-photo.jpg` or edit the image path on the homepage.
